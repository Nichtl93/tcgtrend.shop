# GitHub-Build Schritt für Schritt

1. Öffne dein Repository auf GitHub.
2. Lade den vollständigen Inhalt dieses Projektordners hoch.
3. Achte darauf, dass auch der versteckte Ordner `.github` hochgeladen wird.
4. Öffne oben den Reiter **Actions**.
5. Wähle links **Build Windows RC**.
6. Klicke rechts auf **Run workflow**.
7. Warte, bis der Workflow grün abgeschlossen ist.
8. Öffne den abgeschlossenen Workflow.
9. Lade unten unter **Artifacts** das Paket
   `TCG-Image-Processor-1.0.0-rc1-Windows` herunter.
10. ZIP entpacken und die enthaltene portable Anwendung testen.
